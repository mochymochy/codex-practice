# にゃん缶スライダー デザイン設定

## 1. タイトル・世界観

**正式タイトル：**
にゃん缶スライダー

**サブタイトル：**
〜てんちゃんのお手伝い〜

**テーマ説明：**
ハチワレてんちゃんが倉庫のお手伝いをするよ！

**ゲーム内容：**
てんちゃんを操作して、猫缶を押し、肉球マークの置き場まで運ぶ倉庫番風パズルゲーム。

## 2. てんちゃんのデザイン方針

てんちゃんはハチワレ猫の主人公で、白黒のハチワレ模様を持つ。首まわり・お腹・足先は白、それ以外の体は黒、尻尾は真っ黒。目は丸い白目の中に大きめの丸い黒目とし、全体はシンプルな手書き風・太めの線・フラットな塗りで統一する。

※詳しい設定は `docs/character-design.md` を参照すること。

## 3. 猫缶デザイン

猫缶デザインは、現在のデザイン案を採用する方針とする。

想定用途は以下のとおり。
- 通常の猫缶
- 少し特別な猫缶
- 将来的なコレクション用猫缶
- ステージ目標として運ぶ対象

初期プロトタイプでは、まず通常の猫缶1種類のみを使用する。

## 4. ゲーム画面デザイン

現在のゲーム画面イメージをベースに採用する方針とする。

ただし、以下の点を明記して調整する。
- 初期配置の猫缶の位置は見直す
- ゴールではない角や壁ぎわに猫缶を置きすぎない
- 猫缶が動かせなくなる配置を避ける
- 猫缶を置く場合は、押せる方向とゴールまでの導線を確認する
- タイトル表記は「にゃん缶スライダー 〜てんちゃんのお手伝い〜」に統一する

## 5. UIデザイン

以下を確定方針とする。
- タイトル表記は「にゃん缶スライダー 〜てんちゃんのお手伝い〜」に統一する
- ボタンデザインはカラフルな方を採用する
- アイコン案は現状の方向性を採用する
- ステージ選択パネル案も現状の方向性を採用する
- 全体的に、手書き風・かわいい・見やすい・ブラウザゲームらしい軽さを重視する

## 6. 画像素材の格納ルール

画像素材は以下のフォルダに分けて格納する。

- `assets/characters/tenchan/`：てんちゃん本体の立ち絵・アニメーション差分・表情差分
- `assets/items/`：猫缶やステージ上の運搬対象などのアイテム画像
- `assets/ui/`：ボタン、アイコン、パネルなどUIパーツ
- `assets/logo/`：タイトルロゴや関連ロゴ素材
- `assets/stages/`：床・壁・ゴールなどステージ構成素材
- `assets/references/`：制作時に参照するラフ・比較案・補助資料

## 7. 今後の進め方

1. PNG素材一覧を作る
2. 必要な画像素材を順番に作成する
3. 初期プロトタイプ用の素材を `assets/` に格納する
4. その後、`game/` にブラウザ版の簡単な実装を作る
