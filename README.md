# codex-practice

このリポジトリは、**Codexの練習用**として作成しています。

## このリポジトリの目的
- Codexを使った開発・編集・レビューの練習をする
- 小さなサンプルを通して、操作に慣れる

## 取り扱いルール（重要）
このリポジトリには、次のような機密情報を**絶対に入れないでください**。

- 仕事用データ
- 個人情報
- APIキー
- パスワード

安全に練習するため、公開しても問題ない内容だけを扱います。

## 今後置く予定のコード
- Pythonの練習用コード
- VBAの練習用コード

## 初めての方へ
このREADMEは、初心者の方にも分かりやすいように、
「何のためのリポジトリか」「何を置くか」「何を置かないか」をシンプルにまとめています。

迷ったときは、**機密情報を入れない**ことを最優先にしてください。
